# Getting Started

Stand up a local Toolshed instance, connect an integration, and invoke an operation from the CLI.

## Before you begin

- Google OAuth credentials (or any OIDC provider).
- Node.js 18+ (for the web UI).

## Create a config file

Save this as `toolshed.local.yaml` in the `toolshed/` directory:

```yaml
server:
  port: 8080
  base_url: http://localhost:8080
  dev_mode: true
  encryption_key: local-dev-key

auth:
  provider: google
  config:
    client_id: ${GOOGLE_CLIENT_ID}
    client_secret: ${GOOGLE_CLIENT_SECRET}

datastore:
  provider: sqlite
  config:
    path: ./toolshed.local.db

integrations:
  slack:
    openapi: https://raw.githubusercontent.com/slackapi/slack-api-specs/master/web-api/slack_web_openapi_v2.json
    client_id: ${SLACK_CLIENT_ID}
    client_secret: ${SLACK_CLIENT_SECRET}
    response_check: slack_ok
    auth:
      authorization_url: https://slack.com/oauth/v2/authorize
      token_url: https://slack.com/api/oauth.v2.access
      response_hook: slack_ok
    allowed_operations:
      conversations_list: List channels
      chat_postMessage: Send a message
```

<Callout>
  `base_url` tells Toolshed where to build OAuth callback URLs. If it is wrong or missing, OAuth flows will fail with a redirect mismatch.
</Callout>

## Start the server

```sh
toolshed dev -config toolshed.local.yaml
```

This starts both the API server (port 8080) and the web UI (port 3000). When dev mode is enabled, you will see "dev mode enabled" in the output.

To start only the API without the web UI:

```sh
toolshed dev -config toolshed.local.yaml -api-only
```

## Sign in and connect Slack

<Steps>
### Open the web UI

Go to `http://localhost:3000`. Sign in with your Google account, or use Dev Login in dev mode.

### Connect the integration

Navigate to **Integrations** and click **Connect** next to Slack. Complete the OAuth flow.

### Invoke an operation

Once connected, you can invoke allowed operations from the web UI or CLI.
</Steps>

## Install and use the CLI

```sh
brew install valon-technologies/toolshed/toolshed
tshed init
tshed auth login
tshed integrations list
tshed invoke slack conversations_list
```
