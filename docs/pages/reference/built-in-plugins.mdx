# Built-in Plugins

This page lists the built-in drivers wired into `gestaltd`.

## Auth Providers

| Name | Notes |
| --- | --- |
| `none` | Disables platform auth. |
| `local` | Local single-user login. Supports `config.email`. |
| `google` | Google OAuth login. Supports `client_id`, `client_secret`, `redirect_url`, and `allowed_domains`. |
| `oidc` | Generic OpenID Connect. Supports issuer, client credentials, redirect URL, scopes, PKCE, allowed domains, and display name. |

## Datastores

| Name | Notes |
| --- | --- |
| `sqlite` | Uses `config.path`. Best for local or single-node deployments. |
| `postgres` | Uses `config.dsn`. |
| `mysql` | Uses `config.dsn`. |
| `dynamodb` | Uses `table`, `region`, and optional `endpoint`. |
| `mongodb` | Uses `uri` and optional `database`. |
| `oracle` | Uses `config.dsn`. |
| `firestore` | Firestore-backed datastore. |
| `sqlserver` | Uses `config.dsn`. |

## Secret Providers

| Name | Notes |
| --- | --- |
| `env` | Reads from environment variables. Optional `prefix`. |
| `file` | Reads secret files from a configured directory. |
| `google_secret_manager` | Google Cloud Secret Manager. |
| `aws_secrets_manager` | AWS Secrets Manager. |
| `vault` | HashiCorp Vault. |
| `azure_key_vault` | Azure Key Vault. |

## Telemetry Providers

| Name | Notes |
| --- | --- |
| `stdout` | Structured logs to stdout. Default when omitted. |
| `otlp` | OpenTelemetry logs, metrics, and traces. |
| `noop` | Disable telemetry export. |

## Binding Types

| Name | Notes |
| --- | --- |
| `webhook` | Public or authenticated inbound webhook routes with optional provider forwarding. |
| `proxy` | Authenticated outbound proxy routes with egress policy enforcement. |

## Runtime Types

Gestalt does not register built-in runtime types. Use packaged runtime plugins
under `runtimes.*.plugin`.

## External Packaged Integrations In This Repository

The repository currently ships packaged integration manifests for:

- BigQuery
- Gmail
- Jira
- Notion
- Rippling
- Slack

These are not compiled into the `gestaltd` binary. They are normal packaged
plugins that you reference from `config.yaml`.
