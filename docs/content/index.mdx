# Gestalt

Gestalt is a self-hostable integration platform. A single YAML config defines
how users authenticate, how upstream APIs are connected, and which tools are
exposed through the HTTP API, web UI, and `/mcp`.

## How it works

Gestalt works in three stages:

1. Configure platform auth, storage, secrets, providers, and optional MCP
   naming.
2. Connect users or shared identities to upstream systems through OAuth or
   manual auth.
3. Invoke the same provider catalog through the browser, HTTP API, CLI, or MCP.

## What you can build

- Inline REST providers defined entirely in YAML
- OpenAPI, GraphQL, and upstream MCP integrations
- Executable plugins with Go or Python
- Custom web UI bundles packaged alongside provider releases
