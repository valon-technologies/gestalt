# Gestalt

Gestalt is a self-hostable integration platform. A single YAML config defines
how users authenticate, how upstream APIs are connected, and which tools are
exposed through the HTTP API, web UI, and `/mcp`.

## How it works

You define providers in a YAML config, connect users or shared identities to upstream systems through OAuth or manual auth, then invoke the same provider catalog through the browser, HTTP API, CLI, or MCP. Providers can be inline REST definitions, OpenAPI or GraphQL specs, upstream MCP servers, or executable plugins written in Go or Python.
